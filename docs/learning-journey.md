## How to Use This File

After completing (or making significant progress on) a project, add a new entry below. Include:

- **Date** — when you worked on it
- **Project** — which project number/name
- **What I learned** — new concepts, patterns, or tools you grasped
- **What was tricky** — things that took extra effort or required a breakthrough
- **Next steps** — what you want to explore next based on this experience
