# Project 10: GitHub Profile Viewer

## Description
An app that takes a GitHub username and displays their profile info, repositories, and activity using the GitHub public API.

## Concepts Covered
- GitHub REST API (users, repos, pagination)
- async/await patterns
- Pagination and rate-limit handling
- Responsive card-based layout
- Loading states and error UI

## Suggested Features
- Repository language breakdown chart
- Recent activity feed
- Sort repos by stars, date, language
- Link to open repos on GitHub
