# Project 08: Memory Game

## Description
A card-matching game where players flip cards to find pairs. Tests memory and introduces game-loop programming and animations.

## Concepts Covered
- Game logic (board setup, matching algorithm)
- Timers and move counting
- CSS/Canvas animations for card flips
- Event handling (clicks, keyboard)
- Win/lose state detection

## Suggested Features
- Difficulty levels (3×4, 4×4, 6×6 grids)
- Timer and best-score tracking
- Themed card sets (animals, emojis, icons)
- Sound effects on match/mismatch
