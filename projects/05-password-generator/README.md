# Project 05: Password Generator

## Description
A tool that creates secure, random passwords based on user preferences (length, character types). Great for practicing string manipulation and randomness.

## Concepts Covered
- String manipulation and concatenation
- Random number/character generation
- User options and configuration
- Clipboard or file output

## Suggested Features
- Toggle: uppercase, lowercase, numbers, symbols
- Password strength indicator
- Bulk generation (multiple passwords at once)
- Avoid ambiguous characters option
