# Project 09: Note-Taking App

## Description
A full CRUD note-taking application with persistence. Users can create, read, update, and delete notes, with optional search and rich-text formatting.

## Concepts Covered
- Full CRUD with data persistence (localStorage, SQLite, JSON file)
- Search and filter functionality
- Rich text or Markdown support
- Form validation and confirmation dialogs

## Suggested Features
- Note categories/tags
- Markdown preview
- Export notes to PDF or text
- Dark mode toggle
