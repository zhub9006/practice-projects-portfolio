# Project 14: Chat Application

## Description
A real-time chat app where users can send and receive messages instantly. Introduces WebSocket or Firebase-based real-time communication.

## Concepts Covered
- Real-time communication (WebSocket, Firebase Realtime DB, or Socket.IO)
- Message history and ordering
- User identification (username, avatars)
- Connection state handling (online/offline)

## Suggested Features
- Multiple chat rooms/channels
- Private messaging
- Message timestamps and read receipts
- Typing indicator
