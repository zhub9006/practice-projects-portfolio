# Project 13: Blog Platform

## Description
A simple blog where users can create, edit, and delete posts. Introduces routing, templating, and basic database usage.

## Concepts Covered
- Routing (page navigation without full reloads)
- Templating or component-based rendering
- Database basics (SQLite, MongoDB, or JSON storage)
- Authentication (simple login or admin mode)

## Suggested Features
- Comment system on posts
- Markdown support for writing
- Post categories and tags
- Admin dashboard for managing content
