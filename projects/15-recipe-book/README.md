# Project 15: Recipe Book

## Description
A digital recipe collection where users can add, browse, search, and categorize recipes. A data-focused project that practices modeling, search, and filtering.

## Concepts Covered
- Data modeling (recipes, ingredients, categories)
- Search and filter algorithms
- Image handling (upload or link)
- Category/tag organization

## Suggested Features
- Ingredient-based search ("what can I make with eggs and flour?")
- Recipe scaling (adjust servings)
- Favorite/bookmark recipes
- Print-friendly recipe view
