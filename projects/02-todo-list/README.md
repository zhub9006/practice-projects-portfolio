# Project 02: To-Do List

## Description
A task manager that lets users add, remove, edit, and mark tasks as complete. A classic first project that teaches CRUD operations and state management.

## Concepts Covered
- CRUD operations (Create, Read, Update, Delete)
- State management (keeping track of tasks)
- DOM manipulation or console-based UI
- Data persistence (localStorage, file I/O, or simple database)

## Suggested Features
- Task categories or priority levels
- Due dates and overdue highlighting
- Filter/search tasks
- Drag-and-drop reordering
