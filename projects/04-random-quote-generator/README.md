# Project 04: Random Quote Generator

## Description
An app that fetches and displays random quotes from a public API or a local dataset. A gentle introduction to async programming and working with external data.

## Concepts Covered
- API calls (fetch, requests, or HTTP client)
- Async programming (promises, async/await, callbacks)
- JSON parsing and data display
- Error handling for network failures

## Suggested Features
- Quote categories (inspirational, funny, etc.)
- Copy-to-clipboard button
- Share on social media
- Save favorite quotes locally
