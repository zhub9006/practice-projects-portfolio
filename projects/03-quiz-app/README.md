# Project 03: Quiz App

## Description
A multiple-choice quiz application that presents questions, tracks the score, and shows results at the end.

## Concepts Covered
- Data structures (arrays/objects for questions and answers)
- Scoring logic and conditional branching
- User feedback (correct/incorrect indicators)
- Timer or progress tracking

## Suggested Features
- Multiple quiz categories
- Randomized question order
- Countdown timer per question
- High-score leaderboard
