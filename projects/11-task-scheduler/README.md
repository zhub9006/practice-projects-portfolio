# Project 11: Task Scheduler

## Description
A more advanced task manager that supports due dates, reminders, priority sorting, and recurring tasks. A step up from the basic To-Do List.

## Concepts Covered
- Date/time handling and formatting
- Sorting by priority, date, or category
- Notification or alert system
- Data persistence and serialization

## Suggested Features
- Recurring task support (daily, weekly, monthly)
- Calendar view of tasks
- Overdue task highlighting
- Export schedule to CSV
