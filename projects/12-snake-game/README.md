# Project 12: Snake Game

## Description
The classic Snake game — control a growing snake that eats food and avoids walls and itself. A fun introduction to game-loop programming.

## Concepts Covered
- Game loop (update → render → repeat)
- Canvas or grid-based rendering
- Collision detection
- Scoring and speed progression
- Keyboard event handling

## Suggested Features
- Increasing speed as score grows
- High-score persistence
- Pause/resume functionality
- Different difficulty levels

## Difficulty
🔴 Advanced Beginner / Intermediate
