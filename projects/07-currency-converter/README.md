# Project 07: Currency Converter

## Description
A converter that takes an amount in one currency and returns the equivalent in another, using live exchange-rate data from a public API.

## Concepts Covered
- External API consumption (exchange-rate APIs)
- Real-time data fetching and caching
- Mathematical operations and rounding
- Dropdown/selection UI patterns

## Suggested Features
- Support for 10+ currencies
- Historical rate comparison
- Swap button (flip from/to currencies instantly)
- Offline fallback with last-known rates
