# Project 06: Weather App

## Description
An application that fetches real-time weather data for a city using a public weather API (e.g., OpenWeatherMap) and displays temperature, conditions, and forecasts.

## Concepts Covered
- REST API integration
- JSON response parsing
- Error handling (city not found, API limits)
- Geolocation (optional: auto-detect user location)
- Responsive layout for mobile/desktop

## Suggested Features
- 5-day forecast view
- Weather icons and animations
- Unit switching (°C / °F)
- Search history or saved locations
